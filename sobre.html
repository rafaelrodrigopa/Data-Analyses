File Analysis

O File Analysis é um simples sistema web com a funcionalidade de realizar um comparativo entre dois arquivos de inventário, sendo um arquivo enviado pela empresa contratante e outro pela contratada que realiza as contagens.

Como funciona?

O primeiro input recebe o arquivo final da empresa que realizou as contagens, ele deve estar no formato 24 caracteres por linha, sendo que os primeiros 14 caracteres possui a informação referente ao código do produto e o restante refere-se à quantidade contada. O segundo input recebe o arquivo de cadastro enviado para a empresa contratada realizar as contagens.

Após ter recebido os dois arquivos o sistema irá realizar uma análise para verificar se ambos possuem a mesma quantidade de linhas e vai apurar a quantidade contada, se a quantidade de linhas for diferente o sistema irá acusar incompatibilidade de arquivos.

Há, também, a possibilidade de inclusão da imagem comprobatória da soma total de contagens para comparar com o arquivo. Além disso, o sistema aponta qual o arquivo possui item a mais em relação ao outro.

Pré-requisitos

Dois arquivos de texto, sendo um no formato de 24 caracteres por linha que terá 14 para o código do produto e o restante para a quantidade. O segundo arquivo possui os dados de cadastro de cada produto existente na loja no dia da contagem, sendo que os 14 primeiros devem possuir o código do produto.
